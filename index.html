TwoDotsPlayer

A command-line artificial intelligence algorithm for determining the best moves for the iOS and Android game TwoDots. Utilizes an iterative-deepening depth-first search strategy to optimize space efficiency. Specify your TwoDots game configuration in a YAML file and the number of moves to look ahead, and this algorithm will output a score-maximizing move.


DeckSim

A single-page web application written in HTML/JavaScript/CSS, JQuery, Bootstrap, and Angular. All program data is defined in a JSON data file.

Atlas Of India

A shape grammar (or place grammar) interpreter which takes grammars specified in YAML data files as input, in order to generate a virtual space. The program is multi-screen, graphical, and handles user input. Also includes A* pathfinding and a recursive shadowcasting algorithm.
