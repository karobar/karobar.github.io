TwoDotsPlayer

A command-line artificial intelligence algorithm for determining the best moves for the iOS/Android game TwoDots. Utilizes a depth-first search strategy to optimize space efficiency. Specify your TwoDots game configuration in a YAML file and the number of moves to look ahead, and this algorithm will output a score-maximizing move.

Atlas Of India

A shape grammar interpreter which takes grammars specified in YAML data files as input, in order to generate a virtual space. Shape grammars can be used for procedural design. For some basic information and examples pertaining to shape grammars, check out these slides, courtesy of Carnegie Mellon University. The program displays these grammars using Swing (a release on a Java OpenGL binding is near release) and handles user input. Also includes A* pathfinding and a recursive shadowcasting algorithm.
